# 🚀 Quick Start
📄 Clone or fork `SayyidTestAFX`:
```sh
git clone https://github.com/sayyidisal/SayyidTestAFX.git
```
💿 Install all dependencies:
```sh
cd SayyidTestAFX
yarn install 
```

✏ Create `.env` in the main folder and provide your `GOOGLE_MAPS_API_KEY` and `GEO_CODER` from Google API ([Google Place Autocomplete](https://developers.google.com/maps/documentation/javascript/examples/places-autocomplete)) 
Example:
```jsx
GOOGLE_MAPS_API_KEY = xxxxxxxxxxxx
GEO_CODER = xxxxxxxxx
```

🚴‍♂️ Run your App:
```sh
yarn start
```
